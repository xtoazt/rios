Install Ri0S Profile

Tap the button below on your iPhone or iPad to download and install the Ri0S configuration profile.

Install Profile

How to Install on iOS

  1. Open this page in Safari on your iPhone or iPad.
  2. Tap “Allow” when prompted to download the profile.
  3. Go to Settings > Profile Downloaded (this will appear at the top).
  4. Tap Install in the top-right corner.
  5. Enter your passcode if asked, then tap Install again.
  6. Once installed, the Ri0S profile will be active on your device.

Note: You may need to restart Safari or refresh DNS settings to see the effects.
